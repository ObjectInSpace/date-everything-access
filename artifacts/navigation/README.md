# Navigation Asset Findings

This folder contains asset-mined navigation inputs from the ripped `ThirdPersonGreybox.unity` scene.

## Files

- `thirdpersongreybox-navigation-data.json`
  - Generated by `.\scripts\Export-SceneNavigationData.ps1`
  - Includes serialized `CameraSpaces`, door-like scene objects with world positions, camera objects, and teleporter endpoints

## Regeneration

Run:

```powershell
.\scripts\Export-SceneNavigationData.ps1
```
